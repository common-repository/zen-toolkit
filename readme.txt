== Description ==
Zen Toolkit is a companion for arenabiz WordPress theme by ThemesZen
This plugin adds slider section , about section, portfolio section and testimonial on home page in the arenabiz theme, that adds extra funtionality and features to arenabiz WordPress theme. You need to download and install <a href="https://themeszen.com/arenabiz-theme/" target="_blank">arenabiz</a> theme.

Get free support at https://themeszen.com/contact-us/

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the zen Toolkit plugin through the 'Plugins' menu in WordPress.
